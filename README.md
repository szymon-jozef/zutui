# Usage
### 1. Download:
```
pip install git+https://github.com/shv187/zutui.git
```
### 2. Start:
```
zutui
```
### 3. Enjoy :)

# Bindings
`j` - Down

`k` - Up

`enter` - See partial grades info

`ctrl`+`q` - Quit

# Sidenotes
## Config
- On Unix like is stored @ `~/.config/zutui/config.json`
- On Windows is stored @ `~/zutui/config.json`
- so are `grades_cache.json`
